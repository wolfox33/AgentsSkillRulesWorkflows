# Rules

Regras globais e por agente para padronizar comportamento.

## Estrutura
- `global/`: regras aplicaveis a todos os agentes
- `agents/`: regras especificas por agente

## OpenSpec-first
- OpenSpec e a fonte de verdade para descoberta, spec, plano e criterios.
- Agentes implementam/validam a spec e nao devem duplicar funcoes de SDD.
