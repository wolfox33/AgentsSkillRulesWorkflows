# Agents

Conjunto de agentes para construcao de SaaS, usando as skills em `skills/`.

## Agentes
- product-planner-agent
- architecture-agent
- frontend-agent
- backend-api-agent
- data-auth-billing-agent
- ai-features-agent
- qa-release-agent

Cada agente possui:
- `AGENT.md`: papel, fluxo e limites
- `allowed-skills.md`: skills permitidas para execucao
